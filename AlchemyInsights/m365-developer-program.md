# <a name="microsoft-365-developer-program"></a>Microsoft 365 Geliştirici Programı

Geliştirici Microsoft 365 Programı, kendi korumalı alanlarınızı oluşturmak ve üretim ortamınıza bağımsız olarak çözümler geliştirmek için kullanabileceğiniz bir Microsoft 365 E5 geliştirici aboneliği içerir. Geliştirici Programı ve Abonelikler hakkında daha fazla Microsoft 365 ve SSS Visual Studio için Geliştirici Programı'Microsoft 365 [bakın.](https://docs.microsoft.com/office/developer-program/microsoft-365-developer-program)

Microsoft 365 Geliştirici Programı veya Visual Studio Abonelikler ile ilgili yardım için Abonelik Desteği'ne Visual Studio [gönderin.](https://visualstudio.microsoft.com/subscriptions/support/)
