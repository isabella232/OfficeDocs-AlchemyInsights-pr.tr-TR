# <a name="use-microsoft-intune-to-configure-microsoft-edge-policy-settings-for-windows-10"></a>Windows 10 için Microsoft Edge ilke ayarlarını yapılandırmak üzere Microsoft Intune kullanma

Microsoft Intune'a bir cihaz yapılandırma profili ekleyerek Microsoft Edge ilkelerini ve ayarlarını yapılandırabilirsiniz. Bu, kullanıcı cihazlarında Microsoft Azure Active Directory grup ilkelerini veya yerel Grup İlkesi Nesnesi (GPO) ayarlarını yapılandırmayla eşdeğerdir.

Daha fazla bilgi edinmek için Bkz. [Windows 10 için Microsoft Edge'de ayarları yönetmek](https://go.microsoft.com/fwlink/?linkid=2133700)için profil oluşturma.

Bu makale, Microsoft Edge'in 77 ve sonraki sürümleri için geçerlidir.
