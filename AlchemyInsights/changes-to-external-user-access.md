# <a name="changes-to-external-user-access"></a>Dış kullanıcı erişiminde yapılan değişiklikler

23 Mart 2018'den başlayarak, dış kullanıcılar davet edildi olsalar veya yetkili gruplara eklenmiş olsalar bile artık Herkesle Paylaşılan, Tüm Kimliği Doğrulanmış Kullanıcılar veya Tüm Form Kullanıcıları içeriğine erişmeyecek. Dış kullanıcılar yalnızca kullanıcıyla paylaşılıyor olan içeriğe veya kullanıcının ait olduğu gruplara erişim iznine sahip olur. Mevcut ayarlarınızı tutmak için lütfen aşağıdakilere bakın: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
