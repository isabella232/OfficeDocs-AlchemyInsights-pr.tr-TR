# <a name="make-microsoft-bing-your-default-search-engine"></a>Microsoft Bing'i varsayılan arama motorunuz yapma

Bing'i Microsoft Edge, Google Chrome veya Internet Explorer için varsayılan arama motorunuz yapmak için, Bing'i varsayılan arama alt yapısı yapma [makalesinde verilen yönergeleri izleyin.](https://go.microsoft.com/fwlink/?linkid=2148834)
