# <a name="control-sync-throughput"></a>Eşitleme aktarım hızını denetleme

Eşitleme uygulaması trafiğini denetlemeniz gerekirse mümkün olduğunda ağ hizmet kalitesi (QoS) ilkelerinizi veya Windows QoS ilkelerinizi kullanmanızı öneririz. Kullanıcılarınıza açık olacak ağ ayarları hakkında bilgi için bkz. OneDrive eşitleme uygulamasının karşıya yükleme [veya indirme oranını değiştirme.](https://support.office.com/article/71cc69da-2371-4981-8cc8-b4558bdda56e)
