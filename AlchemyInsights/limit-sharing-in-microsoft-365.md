# <a name="limit-sharing-in-microsoft-365"></a>Microsoft 365'te paylaşımı sınırla

İç paylaşımı tamamen devre dışı bırakamanıza veya sitelerden Paylaş düğmesini kaldıramanıza gerek yoktur, ancak Microsoft 365'te paylaşımı kuruluş ihtiyaçlarını karşılayacak şekilde sınırlandırabilirsiniz. Daha fazla bilgi için bkz. [Microsoft 365'te paylaşımı sınırlama.](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)
